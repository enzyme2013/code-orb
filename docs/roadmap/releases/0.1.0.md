# Release 0.1.0

## Intent

Ship the first believable end-to-end CLI coding agent loop for a local repository.

This release should be built in two phases:

1. runtime contracts and architectural hardening
2. a minimal runnable loop that exercises those contracts on fixture repositories

## In Scope

- CLI entry point
- session, turn, and step runtime definitions
- repository read and search tools
- edit path with auditable mutations
- verification command execution
- structured event stream
- final task report
- minimum safety gate for mutating actions

## Out Of Scope

- desktop shell
- daemon mode
- plugin marketplace
- remote execution
- multi-agent runtime

## Exit Criteria

- can operate on a fixture repository
- can complete at least one representative task from prompt to report
- emits a structured event stream suitable for CLI rendering
- reflects explicit runtime contracts for session, turn, step, tools, and policy
- repository structure still matches the documented architecture
